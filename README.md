## 👋 About This Repository

This is my personal, ongoing DSA practice repo — built over time as I work through core data structures and algorithms.

> **Language note:** I started this journey in **C++**, then transitioned to **Java** as my primary language. All active and new solutions are written in **Java**. Some earlier folders contain C++ code from the initial phase.

This repo is a live record of my learning — not a finished product. Topics are explored progressively and solutions are added as I go.

## 🚀 Getting Started

**Clone the repo:**
```bash
git clone https://github.com/Sukhpreet-Singh2428/DSA-JOURNEY.git
cd DSA-JOURNEY
```

**Run a Java solution:**
```bash
javac FileName.java
java FileName
```

**Run a C++ file (early topics):**
```bash
g++ -o solution FileName.cpp
./solution
```
